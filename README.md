## 🚀 What Is SkillMap?

SkillMap is a full-stack edtech platform designed to disrupt the career development space. Instead of generic advice, it uses **AI to generate personalised, actionable career pathways** for any skill or job profile — in seconds.

### Core Features

| Feature | Description |
|--------|-------------|
| 🤖 **AI Learning Roadmaps** | Enter any skill or job title → get a structured, step-by-step learning roadmap generated by LLMs |
| 📄 **AI Résumé Builder** | Upload your info → AI generates a tailored, ATS-optimised résumé |
| ✉️ **Cover Letter Generator** | Job description in → personalised cover letter out |
| 💼 **Real-Time Job Listings** | Live job feed via API integration — relevant roles updated dynamically |
| 📊 **Industry Insights** | AI-curated insights on skills in demand, salary trends, and hiring patterns |
| 🎯 **Interview Prep** | Role-specific question banks and structured prep guides |

## 🧱 Tech Stack

```
Frontend     Next.js 14 · Tailwind CSS · shadcn/ui
Backend      Next.js API Routes · Prisma ORM
Database     PostgreSQL (via Prisma)
AI Layer     LLM API integrations (OpenAI-compatible)
Jobs API     Real-time job listing feed
Deployment   Vercel (CI/CD via GitHub)
```

## 🔧 Local Setup

```bash
git clone https://github.com/roy030407/skill-map
cd skill-map
npm install
cp .env.example .env.local   # Add your API keys
npx prisma generate
npx prisma db push
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
