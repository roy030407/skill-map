---

## 🚀 What Is SkillMap?

SkillMap is a full-stack AI edtech platform built to disrupt career development. Instead of generic advice, it uses **LLMs to generate personalised, actionable career pathways** for any skill or job profile — in seconds. Every tool on the platform is AI-powered and connected to real-world data.

---

## ✨ Core Features

| Feature | Description |
|--------|-------------|
| 🤖 **AI Learning Roadmaps** | Enter any skill or job title → structured 30–40 step learning path generated by LLMs, from beginner to advanced |
| 📊 **Industry Insights** | AI-powered market growth charts, skills demand distribution, salary trends, and career opportunity analysis per industry |
| 🎯 **Interview Preparation** | Role-specific question banks with instant AI feedback — practice and improve before the real thing |
| 📄 **Smart Résumé Builder** | ATS-optimised résumé generation tailored to user skills and target role |
| ✉️ **Cover Letter Generator** | Job description in → personalised, compelling cover letter out |
| 💼 **Real-Time Job Listings** | Live job feed via API integration — relevant, updated roles matched to user profile |

## 🧱 Tech Stack

```
Frontend     Next.js 14 · Tailwind CSS · shadcn/ui
Backend      Next.js API Routes · Prisma ORM
Database     PostgreSQL (via Prisma)
AI Layer     LLM API integrations for roadmap, insights & content generation
Jobs API     Real-time job listing feed
Charts       Recharts (market growth & skills demand visualisations)
Deployment   Vercel (CI/CD via GitHub)
```

## 🔧 Local Setup

```bash
git clone https://github.com/roy030407/skill-map
cd skill-map
npm install
cp .env.example .env.local   # Add your API keys
npx prisma generate
npx prisma db push
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

**Required environment variables:**
```
DATABASE_URL=
OPENAI_API_KEY=          # or compatible LLM API key
JOBS_API_KEY=            # for real-time job listings
NEXTAUTH_SECRET=
```
